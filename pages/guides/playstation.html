To use the Public GeyserConnect for your Playstation,

you must use a LAN proxy software that relays the GeyserConnect server as a local connection, connecting to either us.geyserconnect.net or eu.geyserconnect.net with port 19132.

iOS Device:

If you have an iOS device, you can use the BedrockTogether app.

Android Device:

If you have an Android device, you can use the GeyserMC Android app. Select Join a Bedrock Server>Enter the GeyserConnect IP>go to Minecraft's Friends tab. If you are not able to install APK's, you can use MC Server Connector.

Computers:

Download and setup Phantom on your computer. You may also run an Android emulator such as Bluestacks to run the apps mentioned above.
